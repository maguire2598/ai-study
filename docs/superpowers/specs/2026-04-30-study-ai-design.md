# 高等数学 AI 复习助手 — 设计文档

## Context

当前大学生普遍缺乏明确的自主学习目标，面对高数等课程不知道如何系统复习。本项目是一个纯前端网站，利用 Mock AI 交互帮助学生制定复习计划、学习知识点、刷题、模拟考试。

## 技术方案

- 原生 HTML/CSS/JS，无框架，无构建工具
- 浏览器直接打开 `index.html` 即可使用
- KaTeX CDN 渲染 LaTeX 数学公式
- localStorage 持久化用户数据（计划进度、错题、收藏）
- 所有 AI 交互使用 mock 数据模拟（`setTimeout` 延迟 300-800ms 模拟 AI 响应）

## 视觉风格

暖色活力风：琥珀/暖橘色调（`#d97706`, `#fef3c7`, `#fffbeb`），圆角卡片，大留白，参考多邻国的亲切感，降低数学畏难情绪。

## 路由设计

Hash 路由，`app.js` 监听 `hashchange`：

| 路由 | 模块 |
|------|------|
| `#dashboard` | 仪表盘主页 |
| `#plan` | 复习计划 |
| `#graph` | 知识图谱 |
| `#learn` | 知识点教学 |
| `#practice` | 习题练习 |
| `#exam` | 考试模拟 |
| `#notebook` | 错题本 & 收藏 |

## 七大模块详情

### 仪表盘主页
- 顶部：考试倒计时 + 复习进度条
- 下方：四宫格导航（知识图谱、每日练习、知识点、模拟考试）
- 今日任务摘要

### 复习计划
- 设置区：考试日期选择、当前水平选择、每日可投入时间
- "AI 生成" 按钮 → loading 动画 → 展示周计划时间线
- 每周计划卡片：时间段、章节、完成状态（✓已完成 / ⏳进行中 / 待开始）

### 知识图谱
- 高数章节以节点依赖图展示（从上到下：函数极限 → 导数 → 积分 → 微分方程）
- 节点颜色：绿（已掌握）、黄（学习中）、灰（未开始）
- 点击节点跳转知识点教学页

### 知识点教学
- 顶部：B站视频嵌入卡片（标题、时长、链接）
- 中部：核心概念列表
- 下部：典型例题 + 分步解析
- 底部按钮：跳转本章习题

### 习题练习
- 顶部：章节筛选标签 + 难度筛选
- 题目卡片：题型标签、难度星级、题干（KaTeX 渲染）
- 选项可点击选择
- 选择后显示 Mock AI 解析结果：正误判断 + 解题步骤 + 关键公式
- 底部：下一题按钮
- 做错的题自动加入错题本

### 考试模拟
- 顶部：倒计时器 + 答题进度（x/20 题）
- 题目区：题型、分值、题干、作答区
- 底部：答题卡导航（绿=已回答，黄=待答，高亮=当前）+ 上下题按钮
- 交卷按钮 + 确认弹窗
- 交卷后：分数展示 + Mock AI 薄弱章节分析

### 错题本 & 收藏
- 章节筛选标签
- 错题列表：题干预览、所属章节、错题次数、日期、重做按钮
- 每道题可点 ⭐ 收藏/取消收藏
- 支持按收藏筛选

## 数据流

1. 用户操作 → 模块 JS 处理 → 读/写 localStorage（进度、错题、收藏）
2. Mock AI 调用 → `data/*.js` 预置数据 + `setTimeout` 模拟延迟
3. 模块间通信 → `app.js` 事件总线（CustomEvent）

## 后续对接真实 API

1. 用户在设置页填入自己的 Claude API Key → 存 localStorage
2. 替换 `data/plan-mock.js` → 调用 Claude API 生成 JSON 计划
3. 替换习题批改 mock → 调用 Claude API 判题 + 解析
4. 替换薄弱分析 mock → 调用 Claude API 分析错题历史
5. API Key 仅存本地，不经过任何服务端

## 高数知识点覆盖（第一版）

函数与极限 → 导数与微分 → 微分中值定理 → 不定积分 → 定积分 → 微分方程 → 多元函数微分 → 重积分

## 验证方法

1. 浏览器打开 `index.html`，确认所有页面正常渲染
2. 各模块路由切换无报错
3. 习题选择后显示 AI 解析（mock）结果
4. 考试模拟倒计时正常、交卷出分正常
5. 错题自动收录、收藏功能正常
6. LaTeX 公式在各模块中正确渲染
7. localStorage 数据刷新后保持
